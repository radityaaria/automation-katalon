# Katalon Web Automation Testing

## Overview
This project contains Web UI automation testing using Katalon Studio.  
The test cases cover login scenarios for valid and invalid credentials.

## Tools
- Katalon Studio
- Selenium-based Web UI Testing
- Groovy
- Git
- GitHub

## Test Scope
- Login with valid credentials
- Login with invalid credentials
- Login without credentials
- Login without password

## Test Cases
| Test Case | Description | Expected Result |
|---|---|---|
| LG_001 | Login with valid credentials | User successfully logged in |
| LG_002 | Login with invalid credentials | Error message displayed |
| LG_003 | Login without input credentials | Required field validation displayed |
| LG_004 | Login without password | Password validation displayed |

## How to Run
1. Clone this repository
2. Open the project using Katalon Studio
3. Open the Test Cases folder
4. Select a test case
5. Click Run
6. Choose Chrome browser

## Result
The login test case was successfully executed and passed in Katalon Studio.
